# Backend API Base (FASE 1)

## Requisitos
- PHP 8.1+
- MySQL 8.x

## Configuracion
1. Copiar `.env.example` a `.env`.
2. Ajustar variables de conexion MySQL.

## Ejecucion local (servidor embebido PHP)
```bash
php -S localhost:8000 -t public
```

## Endpoint disponible en FASE 1
- `GET /api/v1/health`
